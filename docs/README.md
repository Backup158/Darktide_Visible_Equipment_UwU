Addon for the Standalone Visible Equipment mod for *Warhammer 40,000: Darktide*

Adds weapon placements that I think are cool and unserious placements (requires "xd mode" enabled in the Mod Options)
- Weapon Placements

        - for pistols
- Joke

    - Up the butt (flipped)
